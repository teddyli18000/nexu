# Usage & Rewards

Usage quota in nexu covers the model consumption of your Agents. Each time your Agent processes a message through nexu Official using models like GPT, quota is deducted based on the model type and token usage. If you use BYOK (Bring Your Own Key) mode, model costs are charged directly to your API key and no platform quota is consumed.

## Base Quota

Every nexu Cloud user receives a free usage quota upon sign-up:

| Quota Type | Value | Description |
| --- | --- | --- |
| Weekly total | $50 | Resets automatically every 7 days |
| Sliding window | $20 / 5 hours | Prevents burst consumption |

- Quota applies to models provided by nexu Official (GPT, etc.)
- Using BYOK (Bring Your Own Key) does not consume platform quota
- Quota resets at the end of each cycle — unused quota does not roll over

## FAQ

### Do rewards expire?

Rewards count toward your current weekly quota and reset together with it. Unused rewards do not carry over to the next week.

### Can I claim rewards if I use BYOK?

Yes. Reward tasks are available regardless of your usage mode, but reward quota only applies to nexu Official models.

### How is sharing verified?

After sharing, return to the nexu client and tap "Confirm" to claim your reward. We operate on an honor system — no screenshot review is required.

### When does the weekly reset happen?

Every Monday at 00:00 (local time), all weekly task claim statuses are reset automatically.

### What are the benefits of signing in to a Cloud account?

Signing in unlocks the $50 weekly free quota and all reward tasks. Users who are not signed in can still use BYOK mode but will not receive platform quota or rewards.
