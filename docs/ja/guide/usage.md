# 使用量とリワード

nexu の使用量クォータは、Agent 実行時のモデル消費に使用されます。Agent が nexu Official を通じて GPT などのモデルでメッセージを処理するたびに、モデルの種類とトークン消費量に応じてクォータが差し引かれます。BYOK（自分の API Key を使用）モードの場合、モデル費用は API Key に直接請求され、プラットフォームクォータは消費されません。

## 基本クォータ

nexu Cloud ユーザーは登録後、無料の使用量クォータを自動的に受け取ります：

| クォータタイプ | 値 | 説明 |
| --- | --- | --- |
| 週間合計 | $50 | 7日ごとに自動リセット |
| スライディングウィンドウ | $20 / 5時間 | 短時間での集中消費を防止 |

- クォータは nexu Official が提供するモデル（GPT など）の呼び出しに使用されます
- BYOK（自分の API Key を使用）の場合、プラットフォームクォータは消費されません
- クォータは各サイクル終了時にリセットされ、未使用分は繰り越されません

## リワードタスク

以下のタスクを完了すると、追加クォータを獲得できます。リワードは当週の合計クォータに直接加算されます。

### デイリーチェックイン

| 項目 | 詳細 |
| --- | --- |
| リワード | +$1 |
| 頻度 | 1日1回 |
| 方法 | 「もっと使用量を獲得」ページで「チェックイン」をタップ |

- チェックインリワードは週間クォータに累積されます
- チェックイン状態は毎日 0:00（ローカル時間）にリセットされます

### GitHub Star

| 項目 | 詳細 |
| --- | --- |
| リワード | +$3 |
| 頻度 | 1回限り（永久） |
| 方法 | 「受け取る」をタップ → GitHub リポジトリで Star → 戻って確認 |

### ソーシャル共有

各チャンネルは週1回受け取り可能で、毎週月曜日 0:00（ローカル時間）にリセットされます：

| チャンネル | リワード | 方法 |
| --- | --- | --- |
| X (Twitter) | +$2 | 「共有」をタップ → 自動で投稿画面が開く（テンプレート付き）→ 投稿後に戻って確認 |
| Reddit | +$2 | 「受け取る」をタップ → Reddit 投稿ページへ → 投稿後に戻って確認 |
| 小紅書 | +$2 | 「画像を保存」をタップ → 共有カードをダウンロード → 小紅書に投稿 → 戻って確認 |
| LinkedIn | +$2 | 「受け取る」をタップ → LinkedIn 共有ページへ → 投稿後に戻って確認 |
| 即刻 | +$2 | 「画像を保存」をタップ → 共有カードをダウンロード → 即刻に投稿 → 戻って確認 |
| WeChat | +$1 | 「画像を保存」をタップ → 共有カードをダウンロード → モーメンツまたはグループチャットで共有 → 戻って確認 |
| Feishu | +$1 | 「画像を保存」をタップ → 共有カードをダウンロード → Feishu グループで共有 → 戻って確認 |

## リワードまとめ

毎週すべてのタスクを完了した場合：

| ソース | 金額 | 頻度 | 週間最大 |
| --- | --- | --- | --- |
| デイリーチェックイン | $1 | 毎日 | $7 |
| GitHub Star | $3 | 初回のみ | — |
| X 共有 | $2 | 毎週 | $2 |
| Reddit 投稿 | $2 | 毎週 | $2 |
| 小紅書投稿 | $2 | 毎週 | $2 |
| LinkedIn 共有 | $2 | 毎週 | $2 |
| 即刻投稿 | $2 | 毎週 | $2 |
| WeChat 共有 | $1 | 毎週 | $1 |
| Feishu 共有 | $1 | 毎週 | $1 |
| **週間リワード合計** | | | **最大 $19** |

基本クォータ $50 と合わせて、毎週最大 **$69** まで利用可能です（初週は GitHub Star を含めて $72）。

## よくある質問

### リワードクォータは期限切れになりますか？

リワードは当週の合計クォータに含まれ、週次サイクルと一緒にリセットされます。未使用のリワードは翌週に繰り越されません。

### BYOK を使用していてもリワードを受け取れますか？

はい。リワードタスクは使用モードに関係なく利用できますが、リワードクォータは nexu Official モデルにのみ適用されます。

### 共有はどのように確認されますか？

共有操作の完了後、nexu クライアントに戻って「確認」ボタンをタップするとリワードを受け取れます。信頼ベースの仕組みで、スクリーンショットの審査は行いません。

### 週次リセットはいつですか？

毎週月曜日 0:00（ローカル時間）に、すべての週次タスクの受け取り状態が自動的にリセットされます。

### Cloud アカウントにサインインするメリットは？

サインインすると、週 $50 の無料クォータとすべてのリワードタスクが利用可能になります。サインインしていないユーザーは BYOK モードを使用できますが、プラットフォームの無料クォータやリワードは受け取れません。
