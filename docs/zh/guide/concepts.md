# 核心概念

nexu 里有几个关键概念，花两分钟了解一下，后面用起来会更顺手。

## Agent

你的 AI 助手。它能听懂你说的话、帮你干活、还能同时出现在多个聊天软件里。

你可以把它理解成一个 7×24 在线的智能员工。

## 渠道（Channels）

Agent 住在哪里？就是渠道。飞书、Slack、Discord 都是渠道。

连接渠道只需要填几个密钥，Agent 就能在那个平台上收发消息了。一个 Agent 可以同时连多个渠道。

## 模型（Models）

Agent 有多聪明，取决于背后用的模型。

- 想快速上手？用 **nexu 官方模型**，开箱即用
- 想用自己的 Key？支持 Anthropic、OpenAI、Google AI 等主流服务商

随时可以在客户端里切换。

## 技能（Skills）

光聊天还不够——技能让 Agent 能做更多事。

比如查数据库、写周报、操作飞书多维表格……这些都是技能。你可以从技能商店一键安装，也可以自己写一个。

## 部署（Deployments）

打开 nexu、连上渠道，Agent 就自动跑起来了。

不需要买服务器、不需要配 Docker，桌面客户端帮你搞定一切。

## 工作区（Workspace）

工作区就是你在 nexu 里的"家"。Agent、渠道、模型、技能，都归属于同一个工作区。
