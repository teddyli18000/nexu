# 用量与奖励

nexu 的用量额度用于 Agent 运行时的模型消耗。当你的 Agent 通过 nexu Official 调用 GPT 等模型处理消息时，每次请求会根据模型类型和 token 用量扣减相应额度。如果你使用 BYOK（自带 API Key）模式，模型费用直接由你的 API Key 承担，不消耗平台额度。

## 基础用量

每位 nexu Cloud 用户注册后自动获得免费用量额度：

| 额度类型 | 数值 | 说明 |
| --- | --- | --- |
| 每周总额度 | $50 | 每 7 天自动重置 |
| 滑动窗口额度 | $20 / 5 小时 | 防止短时间内集中消耗 |

- 额度用于调用 nexu Official 提供的模型（GPT 等）
- 使用 BYOK（自带 API Key）时不消耗平台额度
- 额度在每个周期结束时自动重置，未使用的额度不累积

## 常见问题

### 奖励额度会过期吗？

奖励计入当周总额度，随每周周期一起重置。未使用的奖励不会累积到下一周。

### 我用 BYOK 还能领奖励吗？

可以。奖励任务不限制使用方式，但奖励额度仅适用于 nexu Official 模型。

### 如何确认分享成功？

完成分享操作后，回到 nexu 客户端点击"确认"按钮即可领取奖励。我们基于诚信机制，不做截图审核。

### 每周重置时间是什么时候？

每周一 0:00（本地时间）自动重置所有每周任务的领取状态。

### 登录 Cloud 账号有什么好处？

登录后即可获得每周 $50 免费额度，并解锁所有奖励任务。未登录用户可以使用 BYOK 模式，但无法获得平台免费额度和奖励。
